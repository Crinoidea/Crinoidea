Про меня

Меня зовут Тамара

Я начинающий web-разработчик из Киева. Прошла курсы по web-разработке и по Java-Script. На данный момент я умею верстать сайты, используя различные технологии, которые ускоряют и облегчают этот процесс. Так же я могу оживить сайт с помощью анимации, простых библиотек, или с помощью кода, написанного на чистом JavaScript. В ближайшее время я планирую более углублённое изучение JavaScript, а так же изучение библиотеки React. Хочу отметить, что данный сайт был сделан мною с помощью GitHub Pages. Так же я буду рада, если вы ознакомитесь с моим портфолио ниже.

Навыки

Что я использую в работе
HTML5

Именно HTML создает каркас сайта или приложения, а пятая версия позволяет создавать более SEO-оптимизированную структуру продукта.

CSS3

CSS – это язык стилей, который позволяет создавать абсолютно любой внешний вид сайта или приложения. Все ограничивается только фантазией!

Java Script

Язык программирования Java Script позволяет оживить все что угодно: слайдеры, окна, подсказки, вкладки, получение данных от сервера и многое другое.

Node.js

Эта платформа позволяет создавать бэкенд для продукта – “мозги”, которые будут выполнять действия, которые пользователь не видит.

jQuery

jQuery – это быстрая, небольшая и богатая возможностями JavaScript библиотека, которая фокурисуется на взаимодействии JavaScript и HTML. Она позволяет ускорить разработку.
